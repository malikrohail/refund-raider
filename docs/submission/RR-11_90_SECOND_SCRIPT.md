# RR-11 90-Second Script

## 0-10 Seconds

"Refund Raider is an AI refund assistant that reads the rules, tells you if you qualify, drafts the request, and sends it for you."

## 10-25 Seconds

Show the problem:

- damaged or wrong item
- merchant support ignored the user
- policy is hard to find

Say:

"You paste the merchant, upload proof, and Refund Raider does the annoying part."

## 25-45 Seconds

Click `Find my refund path`.

Show:

- Firecrawl research
- policy quote
- return deadline
- support email or contact path

Say:

"This verdict is evidence-backed, so the user can see exactly why it qualifies."

## 45-65 Seconds

Open the ElevenLabs agent.

Have it explain the case in plain English:

- what the rules say
- why the user qualifies
- what evidence matters next

Say:

"ElevenLabs is the advocate layer that explains the result and guides the next step."

## 65-80 Seconds

Show the generated refund email.

Say:

"The app drafts the exact request, so the user does not have to write it themselves."

## 80-90 Seconds

Click approve and send.

End on the status timeline:

- research done
- draft ready
- sent
- waiting

Final line:

"Refund Raider turns a confusing refund fight into a concrete action in under a minute."
