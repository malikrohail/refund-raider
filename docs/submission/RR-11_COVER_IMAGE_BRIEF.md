# RR-11 Cover Image Brief

## Objective

Create a thumbnail or cover image that instantly communicates:

- refund problem
- evidence-backed decision
- assistant-driven action
- real send moment

## Visual Concept

Use a split composition:

- left side: messy refund problem, receipt, damaged package, notification bubbles
- right side: clean verdict panel, evidence cards, drafted email, send button

## Must-Have Elements

- Refund Raider title
- one strong hero object, such as a smashed box, wrong item, or receipt
- visible policy/evidence cards
- simple verdict chip like `Qualified` or `Best path: email`
- one clear action cue like `Draft ready` or `Sent`

## Tone

- sharp
- credible
- high-contrast
- practical rather than playful

## Suggested Text on Image

- `Reads the rules. Gets your refund.`
- `Evidence in. Refund out.`
- `AI refund assistant`

## Avoid

- generic AI glow art
- stock chatbot imagery
- too many words
- abstract finance visuals with no refund context

## Layout Guidance

- keep the headline large and readable at small size
- use a single strong focal point
- make the verdict and send state obvious within one glance
