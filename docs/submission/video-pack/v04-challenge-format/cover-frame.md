# v04-challenge-format cover frame

## Headline

Refund In 60 Seconds?

## Focal image

Bold countdown timer paired with one proof artifact and the case page.

## Must-show UI state

- timer
- evidence or verdict
- sent-state or draft-ready cue

## Color and mood

Kinetic, punchy, urgent, higher contrast than the main concept.

## Avoid

Do not let the timer become the whole thumbnail and hide the actual product value.
