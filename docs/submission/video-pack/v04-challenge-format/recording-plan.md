# v04-challenge-format recording plan

## Capture order

1. timer hook
2. merchant/problem/proof card
3. Firecrawl evidence state
4. one short ElevenLabs clip
5. draft state
6. send state
7. timer stop and recap frame

## Mandatory assets

- clean timer overlay space
- one readable evidence frame
- one fast verdict moment
- one recap slate

## Optional assets

- split-screen effect
- fast punch-in crops

## Do not capture

- slow transitions
- extra explanatory clips
