# Refund Raider Video Pack

This folder is the planning-only video pack for the Firecrawl x ElevenLabs submission.

Each concept gets its own identifier and folder so you can choose one path and execute it cold later.

## Structure

- `v01-clean-winner`
  - safest judge-facing concept
- `v02-frustration-to-relief`
  - strongest emotional/social variant
- `v03-voice-magic`
  - ElevenLabs-forward version
- `v04-challenge-format`
  - strongest short-form viral cut
- `v05-policy-detective`
  - builder/judge credibility version

## Recommendation Order

1. `v01-clean-winner`
2. `v02-frustration-to-relief`
3. `v04-challenge-format`
4. `v03-voice-magic`
5. `v05-policy-detective`

## Shared Demo Loop

All five concepts use the same proof loop:

1. problem appears
2. Firecrawl finds evidence
3. ElevenLabs explains the verdict
4. Refund Raider drafts the request
5. user approves and sends
6. timeline reflects the action

Do not turn these into five different products. They are five wrappers around one repeatable hero flow.

## Picking Rule

- If the only goal is to maximize the odds of winning, choose `v01-clean-winner`.
- If you want the strongest emotional short-form cut, choose `v02-frustration-to-relief`.
- If the live voice agent is genuinely excellent, consider `v03-voice-magic`.
- If you want a timer-driven viral cut, use `v04-challenge-format`.
- If you want a more analytical, builder-facing frame, use `v05-policy-detective`.
