# Distribution Playbook

## Default posting order

1. judged submission uploaded first
2. X native video
3. LinkedIn native video
4. TikTok
5. Instagram Reels

## Rule

Do not use the same copy everywhere.

Use:

- `content-engine` for the platform-native angle
- `crosspost` for adaptation logic
- `article-writing` for longer LinkedIn framing or launch notes

## Platform intent

### X

- fastest opener
- least framing
- end on one sharp claim

### LinkedIn

- slightly more context
- emphasize usefulness and why the product matters

### TikTok / Reels

- hardest hook
- shortest caption
- strongest motion

## Best concept mapping

- judged submission: `v01-clean-winner`
- X: `v01-clean-winner` or `v04-challenge-format`
- LinkedIn: `v01-clean-winner` or `v05-policy-detective`
- TikTok / Reels: `v02-frustration-to-relief` or `v04-challenge-format`
