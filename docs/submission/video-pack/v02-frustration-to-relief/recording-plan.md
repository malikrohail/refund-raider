# v02-frustration-to-relief recording plan

## Capture order

1. messy proof or support-thread reality
2. hard cut into Refund Raider
3. Firecrawl evidence state
4. short ElevenLabs explanation clip
5. draft state
6. send state
7. clean resolved end frame

## Mandatory assets

- one messy opening frame
- one clean workspace frame
- one readable best-path result
- one sent-state frame

## Optional assets

- user reaction shot
- cluttered inbox crop

## Do not capture

- too much personal storytelling
- any opening that delays the product reveal
