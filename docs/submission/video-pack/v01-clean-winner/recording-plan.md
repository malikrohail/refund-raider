# v01-clean-winner recording plan

## Capture order

1. final case page with verdict and draft-ready state
2. proof artifact and one-line complaint
3. Firecrawl evidence panel
4. short ElevenLabs explanation clip
5. draft pane
6. approve/send action
7. final timeline state

## Mandatory assets

- readable verdict badge
- readable policy quote
- visible deadline
- visible support path
- readable draft subject/body
- sent timeline event

## Optional assets

- one hero photo of the damaged or wrong item

## Do not capture

- long setup flows
- configuration screens
- browser extension UI
