# v01-clean-winner script

## 0-5s

Voiceover:

`This AI reads the rules, figures out if you qualify, and sends the refund request for you.`

On screen:

- final case state
- verdict visible
- draft ready
- send CTA

## 5-20s

Voiceover:

`Most people lose money because refund policies are buried and support is annoying.`

On screen:

- damaged or wrong-item proof
- merchant name
- short complaint

## 20-40s

Voiceover:

`Refund Raider uses Firecrawl to find the real policy, support path, and deadline, then turns that into a real verdict.`

On screen:

- policy quote
- return deadline
- support email or path
- evidence cards

## 40-60s

Voiceover:

`ElevenLabs is the advocate layer. It explains what the rules mean and what the best next action is.`

On screen:

- short live or captured advocate explanation

## 60-80s

Voiceover:

`Then it drafts the refund request so the user does not have to write it.`

On screen:

- draft subject
- draft body

## 80-90s

Voiceover:

`Evidence in. Refund out.`

On screen:

- approve
- send
- timeline updated
