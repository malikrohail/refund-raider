# v03-voice-magic script

## 0-6s

User says:

`My order arrived damaged and support is ignoring me. Can you handle it?`

Start on voice, not UI.

## 6-25s

Show one tight follow-up question and the case taking shape.

## 25-45s

Show Firecrawl evidence appearing while the call continues.

## 45-65s

Show the agent summarizing:

- whether the user qualifies
- which channel is best
- what happens next

## 65-85s

Show the draft and send confirmation.

## 85-90s

Voiceover:

`The voice agent explains it. The app proves it.`
