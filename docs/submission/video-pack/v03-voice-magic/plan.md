# v03-voice-magic

## Core angle

`Talk to your refund advocate.`

## Best use

Use only if the live ElevenLabs experience is genuinely strong.

## Hook

`I built a voice agent that can look up the rules behind your refund and take the next step for you.`

## Why it works

- makes ElevenLabs feel central
- creates a stronger magic moment
- differentiates the product from a normal web demo

## 90-second beat sheet

### 0-6 seconds

Start on voice, not UI.

User says:

`My order arrived damaged and support is ignoring me. Can you handle it?`

### 6-25 seconds

Show one follow-up question and the case taking shape.

### 25-45 seconds

Show Firecrawl evidence appearing while the conversation continues.

### 45-65 seconds

Show the agent summarizing:

- whether the user qualifies
- which channel is best
- what happens next

### 65-85 seconds

Show draft creation and send confirmation.

### 85-90 seconds

End line:

`The voice agent explains it. The app proves it.`

## Shot list

- `shot-01-hook`: live voice opening with transcript visible
- `shot-02-follow-up`: one precise follow-up question
- `shot-03-firecrawl`: evidence cards appearing behind the conversation
- `shot-04-verdict`: spoken summary plus visible verdict panel
- `shot-05-draft`: draft generated
- `shot-06-send`: send confirmation
- `shot-07-end-card`: timeline and final promise

## Skills plan

### Product

- Firecrawl Search
- ElevenLabs live agent runtime as the primary star

### Video

- `mediabunny` to isolate and clean captured live voice clips
- `elevenlabs-remotion` only for bridge narration
- `elevenlabs-music` only if kept very low
- Remotion for transcript, waveform, and timing overlays
- `video-editing` for voice-first pacing
- `content-engine` for X-first demo language

Medium
