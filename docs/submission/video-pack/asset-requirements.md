# Asset Requirements

These are the common assets needed regardless of concept.

## Required assets

- one hero merchant
- one stable refund scenario
- one proof artifact
- one short complaint summary
- one Firecrawl evidence state with visible source quote and deadline
- one ElevenLabs explanation clip or controlled narration fallback
- one generated draft state
- one send state
- one final timeline state

## Recommended captures

- full-screen desktop capture
- one tight crop of the evidence area
- one tight crop of the draft area
- one tight crop of the sent/timeline state

## Optional assets

- damaged item photo
- cluttered support thread screenshot
- waveform or transcript capture
- timer overlay elements

## Avoid

- noisy tabs
- auth/setup screens
- long scrolling
- browser automation unless it is absolutely required
- any clip where Firecrawl or ElevenLabs looks decorative instead of central
