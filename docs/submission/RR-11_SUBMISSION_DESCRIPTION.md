# RR-11 Submission Description

Refund Raider is an AI refund assistant that helps people recover money when an ecommerce order arrives damaged, wrong, or missing. A user uploads proof, pastes the merchant, and Refund Raider searches the merchant's policy and support paths with Firecrawl, explains whether the user qualifies, drafts the refund request, and sends it by email. The key product promise is simple: the user does not have to become a policy investigator to get a fair refund.

What makes the demo credible is that every verdict is backed by visible evidence. The case page shows the policy quote, deadline, support path, and the generated draft before anything is sent. ElevenLabs powers the conversational advocate layer, so the app can explain the result in plain English and guide the next step without taking over the workflow itself.

Refund Raider is narrow on purpose. It focuses on the most repeatable refund wedge for a live demo: ecommerce, public policy, email-first resolution, and a clear status timeline from research to send. That keeps the experience understandable, useful, and easy to judge in one session.
